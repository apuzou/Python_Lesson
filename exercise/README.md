# Roboter（ロボター）

ターミナル上で動く練習用アプリです。Roboko が名前と好きなレストランを聞き、結果を `ranking.csv` に保存します。

## 概要

- **Roboko** が挨拶し、あなたの**名前**を聞きます。
- 既に `ranking.csv` がある場合は、**カウントの多いレストランから順に**「このレストランは好きですか？ [Yes/No]」と聞きます。
  - **Yes** → そのレストランのカウントを増やして締めの挨拶をし、次のゲストへ。
  - **No** → 次のレストランを聞く。すべて No の場合は「どこのレストランが好きですか？」と聞きます。
- ランキングが空の場合、またはすべて No のときは、**好きなレストラン名**を入力してもらいます。
- 入力されたレストラン名は**パスカルケース**に揃え、既存ならカウントを増やし、新規なら追加して `ranking.csv` に保存します。
- 締めの挨拶のあと、**次のゲスト**の挨拶に戻り、**「exit」と入力するまで**同じ流れを繰り返します。

表示は **Roboko のメッセージが緑**、**あなたの入力は通常の色**です。

## ディレクトリ構成

```
exercise/
├── README.md           # 本ファイル（概要・実行方法）
├── main.py             # エントリポイント
├── requirements.txt    # 依存（termcolor）
├── ranking.csv         # 実行後に生成されるレストランランキング
└── roboter/            # パッケージ
    ├── __init__.py
    ├── conversation.py # 会話メッセージ
    ├── csv_manager.py  # ranking.csv の読み書き
    └── formatter.py    # パスカルケース変換
```

## 実行方法

1. ターミナルで `exercise` ディレクトリに移動する。

   ```bash
   cd exercise
   ```

2. 依存パッケージをインストールする（初回のみ）。

   ```bash
   pip install -r requirements.txt
   ```

3. アプリを起動する。

   ```bash
   python main.py
   ```

4. 名前やレストラン名を聞かれたら入力する。**終了したいときは「exit」と入力**する。

## 終了方法

名前を聞かれたとき、または「どこのレストランが好きですか？」と聞かれたときに **`exit`** と入力すると、プログラムが終了します。
